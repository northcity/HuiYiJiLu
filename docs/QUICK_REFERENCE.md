## 📋 已配置的测试凭证

### 🤖 AI 配置 (DashScope/Qwen)

```
API Key:        <YOUR_API_KEY>
Base URL:       https://dashscope.aliyuncs.com/compatible-mode/v1
推荐模型:        qwen-plus
```

**在应用中**:
1. 打开应用 → ⚙️ 设置 → AI 标签
2. 点击 "复制测试密钥" → 粘贴到 API Key 框
3. 选择模型: Qwen Plus (推荐)

---

```
工作流 App ID:   <YOUR_WORKFLOW_APP_ID>
```

2. 点击 "复制测试ID" → 粘贴到工作流 框

## 🚀 首次使用流程 (3 步)

### 1️⃣ 配置应用 (2 分钟)

```
打开应用
  ↓
点击右上角 ⚙️ 齿轮
  ↓
AI 标签 → 复制测试密钥 → 粘贴
  ↓
选择模型 "Qwen Plus"
  ↓
点击"完成"
```

### 2️⃣ 开始录音 (1 分钟)

```
首屏 → 点击大蓝色按钮
  ↓
说话（录制 30 秒测试）
  ↓
点击"停止"
  ↓
自动保存
```

### 3️⃣ 查看结果 (1 分钟)

```
点击列表中新录音卡片
  ↓
向下滑动查看：
- 📝 转写文本（iOS Speech）
- 🎯 AI 摘要
- 💡 关键要点
- ✅ 行动项
- 📄 (可选) HTML 图文纪要
```

**总耗时**: ⏱️ 4-5 分钟

## 📞 常见问题速查表

| 问题 | 解决方案 |
|---|---|
| ❌ 提示"API Key 无效" | 重新复制测试密钥并重启应用 |
| ❌ 没有生成摘要 | 确认 API Key 已填写，选择了模型 |
| ❌ 没有生成 HTML 纪要 | 完整配置 OSS 信息后重试 |
| ❌ 录音没有开始 | 检查麦克风权限，重启应用 |
| ❌ 转写文字为空 | 最小 5 秒以上录音发言 |

## 💡 使用技巧

### 💡 技巧 1: 快速切换标签页

在设置页面，点击顶部的**任意标签图标**快速切换配置模块。

### 💡 技巧 2: 查看密钥

点击密钥输入框右侧的 👁️ 图标显示/隐藏 API Key。

### 💡 技巧 3: 一键复制

在 AI/工作流/听悟 标签中，直接点击"复制测试密钥"或"复制测试ID"按钮。

### 💡 技巧 4: 官方链接

点击 footer 中的链接 "获取密钥 →" 直接打开浏览器。

### 💡 技巧 5: 长期录音

如需录音超过 1 小时，建议使用"Qwen Long"模型。

## 🆘 紧急求助

### 应用无法启动

```
尝试:
1. 强制关闭应用 (从后台滑掉)
2. 在设置 → 存储空间中清除缓存
3. 重新打开应用
```

### 设置页无法打开

```
尝试:
1. 检查网络连接
2. 检查麦克风权限
3. 重启设备
4. 重新安装应用
```

### 无法保存配置

```
尝试:
1. 确保每个字段不为空
2. 检查 App Group 权限
3. 强制刷新页面 (下拉)
```
