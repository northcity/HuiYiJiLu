## 二、视觉系统

### 品牌色系

**主色** — 天空蓝 (Confidence & Sky)
- HEX: `#0084FF`
- RGB: (0, 132, 255)
- 用途: 主按钮、导航、关键交互

**辅色** — 云雀金 (Elegance & Accent)
- HEX: `#FFB500`
- RGB: (255, 181, 0)
- 用途: 高亮、重点提示、成就感

**背景色** — 极简白 (Clean & Modern)
- HEX: `#FFFFFF` 或浅灰 `#F8F9FB`
- 用途: 主背景、卡片

**文字色** — 深空灰 (Readable & Professional)
- 主文本: `#1A1A1A`
- 次文本: `#666666`
- 禁用文本: `#CCCCCC`

### App 图标设计建议

```
设计元素:
├── 核心图形: 飞翔的雀鸟 + 麦克风混合元素
│   └── 雀鸟身体: 记录的信息波形
│   └── 雀鸟翅膀: 向上飞翔的动态感
├── 背景: 渐变天蓝色 #0084FF → #4DB8FF
├── 风格: 现代极简 + 3D 感
└── 安全区域: 中心 60% 区域可视

建议图标变体:
- App Store 图标 (1024x1024)
- Spotlight 图标 (120x120)
- Settings 图标 (58x58)
- HomeScreen 图标 (180x180)
```

### 颜色应用场景

| 界面元素 | 颜色 | 场景 |
|---|---|---|
| 主按钮（开始录音） | 天空蓝 `#0084FF` | 大面积 CTA |
| 灵动岛提示 | 天空蓝 + 云雀金 | 实时录制指示 |

## 四、UI 组件规范

### 主要按钮状态

**开始录音按钮 (CTA)**
```
默认: 天空蓝 (#0084FF), 圆形, 对半径
悬停: 深蓝 (#0066CC), 略微缩放
按下: 深蓝 + 震动反馈
禁用: 浅灰 (#CCCCCC), 不可交互
```

**卡片设计**
```
背景: 白色 / 浅灰 #F8F9FB
圆角: 12px
阴影: 轻微阴影 (0, 2px, 8px, rgba(0,0,0,0.08))
选中: 蓝色边框 2px + 浅蓝背景淡化
```

### 状态指示器

| 状态 | 图标 | 颜色 | 动画 |
|---|---|---|---|
| 🔴 正在录制 | 脉冲圆圈 | 红色 `#FF3B30` | 1s 脉冲 |
| ⏸️ 已暂停 | 暂停符号 | 橙色 `#FF9500` | 静止 |
| ✅ 已完成 | 对勾 | 翠绿 `#34C759` | 出现动画 |
| ⏳ 处理中 | 加载动画 | 天空蓝 `#0084FF` | 旋转 |
| ⚠️ 错误 | 感叹号 | 红色 `#FF3B30` | 摇晃 |
